#  前端性能优化


::: tip
用户输入输入URL到页面渲染经历了什么？？
:::

**潜在的性能优化点**

1. DNS是否可以通过缓存减少DNS查询时间
2. 网络请求是否可以走最近的网络环境
3. 相同的静态资源是否可以缓存
4. 是否可以减少HTTP请求大小
5. 减少HTTP请求
6. 服务端渲染



## 资源合并压缩

1. 减少HTTP请求数量
2. 减少请求资源的大小

### HTML压缩
压缩文本文件中有意义，在HTML中没有意义的文本，空格，回车等

**方式**

1. 在线网站压缩
2. node提供HTML-minifier 工具
3. 后端模板引擎渲染压缩

### CSS压缩

1. 无效代码删除
2. 相同代码合并
3. clear-css

### js压缩 

1. 无效字符删除
2. 剔除注释
3. 代码语义的缩减和优化
4. 代码保护
